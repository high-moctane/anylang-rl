## 使い方

言語の各フォルダで

```
make
```

でいろいろビルドとかします。

```
make run
```

で実行時間を計りつつ実験をします。

```
make images
```

で実験結果をプロットしたりします。

各言語についてこれらをやるのはだるいので，`make_caller` スクリプトを書きました。
使い方は，

```
./make_caller run
```

とかそんな感じです。
`-j` オプションで並列実行数を指定できます。
